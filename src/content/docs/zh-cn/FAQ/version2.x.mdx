## 2.x 版本的代码

几乎进行了重写, 主要包括:

- 富文本编辑器改用 [Jodit](https://xdsoft.net/jodit/)
- vue 改用 react
- vite 改用 rspack(为了打包后 Jodit 不报错)

### 关于富文本编辑器

我们尝试过以下编辑器:

- [editor.js](https://github.com/codex-team/editor.js)
- [Jodit](https://github.com/xdan/jodit)
- [Lexical](https://github.com/facebook/lexical)
- [Mdx editor](https://github.com/mdx-editor/editor)
- [Quill](https://github.com/quilljs/quill)

我们希望使用 HTML 保存用户数据, 因为 HTML 通用性强，且数据量更小.

最终我们决定创造一个新的编辑器,叫做 **exsied**, [Github repo](https://github.com/exsied/exsied) / [Gitee repo](https://githee.com/exsied/exsied) / [Document](https://enassi.pages.dev/en/exsied/about/).

## 2.x 版本的功能

功能有所调整:

- 去掉了文件保险箱，可以使用 **导入文件并加密** 实现一样的功能。
- 暂时去掉了分享给朋友

## 从 1.x 版本迁移

由于差异过大，需要逐个复制 1.x 版本里的每个文件到 2.x 版本。
