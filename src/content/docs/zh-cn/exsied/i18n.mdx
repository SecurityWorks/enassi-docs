## 如何自定义 locale?

### 第1步: 使用 `exsied.setDict` 设置字典.

```js
exsied.setDict("zh-CN", {
  Title: "标题",
  Alternative: "别名",
  Styles: "样式",
  Width: "宽度",
  Height: "高度",
});
```

### 第2步: use `exsied.setLocale` 设置 locale.

```js
exsied.setLocale("zh-CN");
```
