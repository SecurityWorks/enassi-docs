目前，**已存在**仅支持英语(en)、法语(fr)、德语(de)、俄语(ru)、西班牙语(es)、简体中文(zh-Hans)和繁体中文(zh-Hant)。

您可以在`exsied.init`中添加`locale`参数以设置默认区域设置，也可以使用以下函数设置新的区域设置。

**注意**：在`exsied.init`之后更改区域设置将需要重新初始化**exsied**,开发者应首先备份内容。

### 添加其他 locale

#### 第 1 步: 找到所有单词

在 IDE 中搜索 `t('`

#### 第 2 步: 设置字典

```js
exsied.i18n.setDict("zh-CN", {
  Title: "标题",
  Alternative: "别名",
  Styles: "样式",
  Width: "宽度",
  Height: "高度",
});
```

#### 第 3 步: 设置 locale

```js
exsied.i18n.setLocale("zh-CN");
```
