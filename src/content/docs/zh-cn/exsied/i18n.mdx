title: 国际化
description: ""
---

目前, **exsied** 只支持英语. 开发者可以根据一下步骤定制自己的语言:

### 第 1 步: 找到所有单词

在IDE中搜索 `t('`

### 第 2 步: 设置字典

```js
exsied.i18n.setDict("zh-CN", {
  Title: "标题",
  Alternative: "别名",
  Styles: "样式",
  Width: "宽度",
  Height: "高度",
});
```

### 第 2 步: 设置 locale

```js
exsied.i18n.setLocale("zh-CN");
```
