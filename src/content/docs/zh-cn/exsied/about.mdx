**Exsied**: **Ex**tremely **si**mple **ed**itor. 发音是 `/ɪkˈsiːd/`, 和 **exceed** 一样

[github repo ](https://github.com/exsied/exsied) / [gitee repo ](https://gitee.com/exsied/exsied)

## 特性:

- 无复杂概念, **exsied** 完全使用 JavaScript 原生的事件绑定

- 无依赖

- 易于配置 / 自定义 / 开发

- 所有功能都基于插件
