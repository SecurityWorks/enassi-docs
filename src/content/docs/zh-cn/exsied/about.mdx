**Exsied**: **Ex**tremely **si**mple **ed**itor. 发音是 `/ɪkˈsiːd/`， 和 **exceed** 一样.

**Exied** 是 [Fivim](https://github.com/fivim/fivim/) 的主编辑器. **Fivim** 是您的加密助手， 支持多种文件类型(包括标记、源代码、PDF、图像等)，支持文件加密和同步.

**Exied** 提供了各种插件，基本上可以开箱即用，但也有一些插件需要配置回调函数才能获得更好的用户体验，如**sourceCode**、**redoAndUndo**、**fontSize**、**font Family**.

[文档](https://fivim.top/zh-cn/exsied/about/)

[Github 仓库](https://github.com/exsied/exsied) / [Gitee 仓库](https://gitee.com/exsied/exsied)

[Live demo](https://exsied.pages.dev/demo)

## Features:

- 无复杂概念， **exsied** 完全使用 JavaScript 原生的事件绑定

- 无依赖

- 易于配置 / 自定义 / 开发

- 所有功能都基于插件
