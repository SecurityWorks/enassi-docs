## How to custom my locale?

### Setp1: use `exsied.setDict` set a dict.

```js
exsied.setDict("zh-CN", {
  Title: "标题",
  Alternative: "别名",
  Styles: "样式",
  Width: "宽度",
  Height: "高度",
});
```

### Setp2: use `exsied.setLocale` set a locale.

```js
exsied.setLocale("zh-CN");
```
